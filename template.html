Les types simples

  • les "ints"
    • aka "les entiers"
    • 1, 2, 4, ..., 10, ...
  • les "float"
    • aka "les nombres à virgules"
    • 1.1, 10.42, .2, 0.3
  • les "strings"
    • aka "les chaînes de caractères"
    • "bonjour", "bière", "a"

Les listes

  • contient une "liste" de n'importe quoi
    • [1, "a", 42.3]
    • lettres = ["a", "b", "c"]
  • accès:
    • ["a", "b", "c"][0] → "a"
    • commence à ZERO et pas à 1 !
    • ["a", "b", "c"][1] → "b"
  • modification:
    • lettres = ["a", "b", "c"]
    • lettres[0] = "pouet"
    • lettres → ["pouet", "b", "c"]

Les strings comme des listes

  • c'est une liste d'éléments d'une lettre
    • "abc"[0] → "a"
    • "abc"[1] → "b"
  • concatenation :
    • "abc" + "pouet" → "abcpouet"

Les tuples, des listes non modifiables

  • comme une liste avec des () à la place des []
    • ("a", "b", "c")
    • (0, 1, 2)[0] → 0
  • impossible de les modifiers (pas important normalement pour vous)

Les dictionnaires

  • représente une correspondance 1 à 1
    • {"a": 1, 3.14: "pi"}
  • accès:
    • {"a": 1, "b": 42}["b"] → 42
    • {"a": 42, "b": 3}[1] → erreur
  • modification:
    • lettres = ["a", "b", "c"]
    • lettres[0] = "pouet"
    • lettres → ["pouet", "b", "c"]

Combinaisons

    a = [(1, 2, 3), 42, "pouet"]

    b = [[(1, 2), (3, "pouet")]}, [1, 2, 3]]
